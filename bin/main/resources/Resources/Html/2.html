You can select the immediate neighbours by pressing Ctrl-N and selecting recursively all neighbours by pressing Ctrl-Shift-N. In the same way you can select children of a node by pressing Ctrl-Alt-C or all children by pressing Ctrl-Alt-Shift-C. In the same manner you can select immediate parents by pressing Ctrl-Alt-P or recursively all parents Ctrl-Alt-Shift-P.

For selecting all visible nodes in the graph press Ctrl-A.
